Exercise 1: Setting up your development environment


Before you perform the exercises for this course, you must do the following:

  1. Downloaded the Yelp dataset. (Note: This database is 1G and will take a while to download) https://s3.amazonaws.com/neo4j-sandbox-usecase-datastores/v3_5/yelp.db.zip

  2. Unzip the zip file. It will create a yelp.db folder.

  3. In Neo4j Desktop, create a project and a local graph, providing a password you will remember. Note: Do not start the database! .Install the APOC plugin in the project. .Install the Graph Algorithms plugin in the project. .Install Graph Algorithms Playground in the project. .Copy the yelp.db folder to your local graph:

    1. Click the Manage button for the local graph you just created.

    2. Click the Open Folder button.

    3. Navigate to data/databases.

    4. Copy the yelp.db folder that you unzipped to databases.

    5. Rename the yelp.db folder to graph.db.

  4. Start the database.

  5. Open Neo4j Browser for the project.

This is what you should see when you click the database icon .

Go to the next page to continue setup.

Verify libraries and graph apps are ready


Verify that your Neo4j Browser session has access to the apoc and algo libraries by executing the Cypher below:

CALL dbms.procedures()
YIELD name
WHERE name STARTS WITH 'apoc.' OR name STARTS WITH 'algo'
RETURN name ORDER BY name ASC

 

If this code does not return the list of apoc and algo procedures, then you must ensure that the these libraries are available by installing the plugin (Neo4j Desktop) and restarting the database.
