About the Course

In this course you will learn how to use graph algorithms in Neo4j. This course takes an applied approach - covering not just how to use the graph algorithms in Neo4j, but also how to leverage graph algorithms to enhance the functionality of a web application.

Topics Covered

  • The Neo4j Graph Algorithms Plugin

  • Personalized PageRank

  • Similarity metrics

  • Community detection

  • Using all of the above to add functionality in an application such as personalization, recommendations, and enhancing search results.</li>
